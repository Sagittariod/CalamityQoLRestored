Re-introduces many different features Calamity used to have which gave power to the player through various means. Up to date with the latest Calamity version.

Mod icon by Zenovia.


>- Overview -<

- Reintroduced many removed items from various Town NPC shops
- Restored recipes for a plethora of vanilla items
- Reintroduced removed items from the Starter Bag's former loot pool
- Expert-exclusive drops are once again obtainable on all difficulties
- Various item drop rates from enemies are increased
- Angler quest rewards have been increased substantially
- Reintroduced removed items from the loot pools of multiple Calamity biome crates
- Fruits drop from trees at an increased rate
- Made Cosmolight obtainable in early hardmode

This mod introduces other optional changes not included by Calamity in any version, all of which default to being off. These are listed below.

- Added the ability for non-legendary weapons dropped by a given boss to be shimmered into one another
- Added a recipe for Bakidon to be obtainable late into pre-hardmode
